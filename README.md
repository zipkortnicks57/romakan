# ROMAKAN 

Simple python library for converting romaji to kana

## Usage
Import romakan to your project:  

`from romakan import Romakan`  

Enjoy with converting romaji to kana:  

```
    converter = Romakan()  
    result = converter.convert_hiragana("syasin")  
    print(result) 
```

Will print:  

```しゃしん```
